About

This is Josh Huang's blog. It's my intention to write something useful.


I created this website from scratch. It's hosted on my GitHub repository. I can edit, push, and make changes live all for free. This website initially started as a place where I could place my answers to computer programming problem sets. It evolved to a blog, and now I'm using it to gauge interest in new projects I'm creating. If you would like me to continue to create products I preview here, please join the waitlist!

If you are interested in creating something similar, I recommend checking out Harvard's CS50: Introduction to Computer Science to understand programming basics. The course was recently featured in the New Yorker. From there follow this link to GitHub Pages to build your website for free.
